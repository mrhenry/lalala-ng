# Haraway

## How to use

1. Create initializer (config/initializers/haraway.rb)
2. Add "haraway_metadata" column to the necessary tables (type = "text")
3. In the model:  
`has_many_assets :asset_association_name, :profile`
`attr_accessible :asset_association_name`
